# 2026-02-12 新增通用调研工具集 (Universal Research Tools)

## 更新内容
在 `digeino` 的核心工具库中新增了 `research` 模块，旨在为所有基于 DigEino 的 Agent 提供强大的自动化调研能力。

### 新增工具
1.  **research_grep**：在代码仓库或文档库中进行深度关键词检索。
2.  **research_read**：精确读取指定文件的全部内容。
3.  **research_doc_to_md**：支持将 PDF/Word 等多种文档格式实时转换为纯 Markdown 文本。
4.  **firecrawl_scrape**：集成 Firecrawl 引擎，实现对任意网页的深度爬取及 Markdown 格式化处理。

### 架构优化
- 将上述工具集成至 `BaseTools` 注册函数，实现一经引入、全局自动加载的效果。
- 采用扁平化工具设计，支持 Agent 按需在编排时灵活绑定。

## 影响面
- 影响 `github.com/originaleric/digeino/tools` 包。
- 提升了 Agent 对非结构化数据（源码、网页、PDF）的处理深度。
