### 2026-02-12 (v1.0.3) 新增通用调研工具集与组件配置增强
- **核心工具集**：新增 `research_grep`, `research_read`, `research_doc_to_md`, `firecrawl_scrape`。
- **文档深度解析**：`research_doc_to_md` 升级为正式版，集成 **Unstructured.io API**，支持 PDF/Docx 高质量 Markdown 转换。
- **WebSearch 集成**：新增 `web_search` 工具，支持 Bocha 和 SerpApi 引擎。
- **配置化增强**：`firecrawl_scrape` 与 `web_search` 现在支持通过配置文件（如 `eino.yml`）进行密钥和参数管理。
- **结构优化**：新增 `ToolsConfig` 统一管理三方工具配置。

## 配置说明 (Configuration)

在您的配置文件（如 `eino.yml`）中，可以进行如下配置：

```yaml
Tools:
  Firecrawl:
    ApiKey: "您的 API Key"
  WebSearch:
    Engine: "bocha" # 支持 bocha, serpapi
    ApiKey: "您的 API Key"
    BaseUrl: "" # (可选) 覆盖默认 API 地址
  Unstructured:
    ApiKey: "您的 Unstructured API Key"
    BaseUrl: "https://api.unstructured.io/general/v0/general"
```

## 注意事项
在 `digeino` 的核心工具库中新增了 `research` 模块，旨在为所有基于 DigEino 的 Agent 提供强大的自动化调研能力。

### 新增工具
1.  **research_grep**：在代码仓库或文档库中进行深度关键词检索。
2.  **research_read**：精确读取指定文件的全部内容。
3.  **research_doc_to_md**：支持将 PDF/Word 等多种文档格式实时转换为纯 Markdown 文本。
4.  **firecrawl_scrape**：集成 Firecrawl 引擎，实现对任意网页的深度爬取及 Markdown 格式化处理。

### 架构优化
- 将上述工具集成至 `BaseTools` 注册函数，实现一经引入、全局自动加载的效果。
- 采用扁平化工具设计，支持 Agent 按需在编排时灵活绑定。

## 影响面
- 影响 `github.com/originaleric/digeino/tools` 包。
- 提升了 Agent 对非结构化数据（源码、网页、PDF）的处理深度。
