### 2026-02-12 (v1.0.3) 通用调研工具集、组件配置增强与向量搜索集成
- **语义检索**：新增 `research_semantic_search` 工具，支持基于向量相似度的代码与文档查找。
- **项目索引**：新增 `research_code_index` 工具，支持一键扫描项目并建立 Pinecone 语义索引。
- **文档深度解析**：`research_doc_to_md` 集成 **Unstructured.io API**，支持 PDF/Docx 高质量 Markdown 转换。
- **核心工具集**：新增 `research_grep`, `research_read`, `firecrawl_scrape`。
- **配置化增强**：支持 `Firecrawl`、`Unstructured`、`Pinecone` 和 `Embedding` 模型的标准化配置。

## 配置说明 (Configuration)

在您的配置文件（如 `eino.yml`）中，可以进行如下配置：

```yaml
Tools:
  Firecrawl:
    ApiKey: "您的 API Key"
  Unstructured:
    ApiKey: "您的 Unstructured API Key"
    BaseUrl: "https://api.unstructured.io/general/v0/general"
  Pinecone:
    ApiKey: "您的 Pinecone API Key"
    Host: "您的 Pinecone Host"
    Namespace: "索引命名空间"
  Embedding:
    Type: "qwen" # 或 openai
    ApiKey: "您的 Embedding API Key"
    Model: "text-embedding-v2"
```

## 注意事项
在 `digeino` 的核心工具库中新增了 `research` 模块，旨在为所有基于 DigEino 的 Agent 提供强大的自动化调研能力。

### 新增工具
1.  **research_grep**：在代码仓库或文档库中进行深度关键词检索。
2.  **research_read**：精确读取指定文件的全部内容。
3.  **research_doc_to_md**：支持将 PDF/Word 等多种文档格式实时转换为纯 Markdown 文本。
4.  **firecrawl_scrape**：集成 Firecrawl 引擎，实现对任意网页的深度爬取及 Markdown 格式化处理。

### 架构优化
- 将上述工具集成至 `BaseTools` 注册函数，实现一经引入、全局自动加载的效果。
- 采用扁平化工具设计，支持 Agent 按需在编排时灵活绑定。

## 影响面
- 影响 `github.com/originaleric/digeino/tools` 包。
- 提升了 Agent 对非结构化数据（源码、网页、PDF）的处理深度。
