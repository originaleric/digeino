# 2026-02-21 更新：重构网页搜索模块

## 更新内容

1. **模块化 `web_search` 工具**
   - 提取了统一的 `SearchProvider` 接口 (`Search`, `Name`)。
   - 将各个搜索引擎（`Bocha`, `Google`, `Bing`, `DuckDuckGo`, `SerpAPI`）的具体实现拆分为独立文件。
   - 利用 `web_search.go` 统一分发请求，实现与旧版 `web_search` 工具无缝对接，并在灵活性和扩展性上对标 DigFlow。
2. **优化 Bocha 搜索结果处理**
   - 调整了 `bocha` 提供者的字段解析，优先返回更详细的 `Summary` 字段，只有在缺失时才降配使用 `Snippet` 字段。
   - 修复了因为 `web_search` 包重构导致的 `semantic_search.go` 中的 `SearchResult` 定义缺失问题。
3. **消除第三方日志依赖**
   - 移除了被移植代码中对 `x/zap` 和 `DigFlow/internal/utils/logger` 的依赖。
   - 统一转用标准库 `fmt` 配合 DigEino 本身的错误处理机制来打印和反馈执行日志。
