## 新增 LLM 工具

- **`write_preview_manifest`**：写入 `preview-manifest.json`（`kind`、`entry`、`assets`、`editable_model`、`revision` 等）。
- **`apply_preview_patch`**：在 `base_revision` 与当前 manifest 一致时应用 `patches`（`html_text` / `html_attr` / `html_inner` / `json_pointer` / `literal_replace`），成功后 `revision++`。
- 支持 `manifest_path` 或 `artifact_id` 二选一（`manifest_path` 优先）。
- 默认在写入前保存历史快照（可配置关闭）。
- **`export_preview_bundle`**：将 manifest 涉及文件打成 zip 并写入工作区/会话目录。

## 程序化 API（供宿主直接调用）

- `ui_ux.ApplyPreviewPatches(ctx, manifestRel, baseRevision, patches)`
- `ui_ux.BuildPreviewZIP(ctx, manifestRel)`、`ui_ux.WritePreviewZIPToFile(ctx, manifestRel, zipRelPath)`

## 配置示例

```yaml
UIUX:
  TempStorage:
    BaseDir: "storage/temp"
  Preview:
    MaxPatchFileBytes: 0   # 0 表示默认 5MiB
    AllowedExtensions: [".tsx", ".jsx", ".ts", ".js", ".html", ".htm", ".json", ".css", ".md"]
    HistoryEnabled: true
    HistoryDir: "preview/history"
```
